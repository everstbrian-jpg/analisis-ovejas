# Análisis de salud y alimentación en ovejas

## Problema
Analizar cómo el tipo de alimentación influye en el estado de salud y peso de las ovejas.

## Datos
Dataset simulado con información de:
- Edad
- Peso
- Tipo de alimento
- Enfermedades
- Frecuencia de alimentación

## Proceso
1. Limpieza de datos
2. Análisis exploratorio
3. Agrupación por tipo de alimento
4. Visualización de resultados

## Resultados
- Se identificó qué tipo de alimento genera mayor peso promedio
- Se observaron patrones de enfermedades según alimentación

## Herramientas
- Python
- Pandas
- Matplotlib

## Conclusión
El tipo de alimentación tiene impacto directo en el peso y posiblemente en la salud de las ovejas.
